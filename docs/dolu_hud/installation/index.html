Installation

If you want to build the project yourself, you can refer to Development

Dependencies​

  • pma-voice
  • ox_lib
  • ox_core
  • ox_inventory

Download & Install​

  • Download the latest release from here,
  • Unzip dolu_hud.zip file,
  • Put dolu_hud folder into your resources folder,
  • Add dolu_hud to your server.cfg file, under all dependencies:
  ensure pma-voice
  ensure ox_lib
  ensure ox_core
  ensure ox_inventory
  ...
  ensure dolu_hud
